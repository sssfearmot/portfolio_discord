## 기능
- Firebase 데이터베이스를 사용한 로그인/로그아웃
- Firebase 데이터베이스를 사용한 채팅
- 채팅방 개설

반응형 웹사이트</br>
Spline을 사용하여 3D 모델 출력

## 사용된 기술
- ReactJS
- Tailwind
- Firebase
- Webpack
- ESLint
- Spline
- Babel
- Sass
